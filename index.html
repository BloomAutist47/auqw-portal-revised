What is Grimlite?

Grimlite is a combination of Grimoire 1.7.8+ (Legacy Version), AQLite , and a bunch of new updates and bugfixes.

How to install Grimlite (Latest Version)

  1. Create empty folder. Rename it to Grimlite.
  2. Open `GrimLite_Updater.rar`
  3. Extract `updater.exe` to the empty folder.
  4. Run the `updater.exe`. It will download the latest Grimlite
  5. Run Grimoire.exe

    6. Note : To update Grimlite, simply run the `updater.exe` once more. It'll download the latest version from github.

What is Grimoire 1.7.8+?

Grimoire is the original client that was started by Biney. When he reached the 3.8 version, it was passed on to one developer to another and improved upon.

How to install Grimoire 3.8 PRO+ 1.7.8 (Legacy Version)

  1. Create empty folder. Rename it to Grimoire.
  2. Open `grim-3.8-pro-1.7.8-eol-patched.zip`
  3. Extract contents to the empty folder.
  4. Run Grimoire.exe

What is AQLite?

"AQLite is a client made for AQWorlds that provides all sorts of Quality of Life tools that have been longed for. With the approach of the new versions of AQWorlds (AQW2 and AQW:Mobile), I am releasing AQLite to help players wait for the forever-waited changes to the game. AQLite is open-sourced to assure that there is no account-stealing, or any funny business contained in the client!"

- 133Spider

Note : AQLite and it's creator has no connection to grimoire and other related grim boating matters.
